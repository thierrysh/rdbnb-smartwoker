Ibitwerekeye

Kuki Ukwiye kutwizera? Menya ibyacu!

RideBNB ni application cyangwa se program yubatswe muburyo bwa Decentrilised applications, aribyo mumagambo ahinnye byitwa DAPS, DAP Ni program ishingiye ku ikoranabuhanga rya Block Chain, Block Chain akaba ari uburyo bugezweho mukubika amakuru muburyo budashobora kwangizwa cyangwa kuvogerwa bitewe nuko ikora muburyo bwa SmartContract cyangwa se amasezerano y'Ubwenge

Ibikorwa bibera muri block chain ni ibikorwa byikora, bidakeneye umuntu ubigenzura, DAPS akaba ari application ikora muburyo bwa SmartContract, aho ibikorwa biberamo biba bidafite umuntu ubigenzura cyangwa ikigo kibigenzura, kuburyo nyirabyo cyangwa ikigo kibigenzura cyabitwara uko byishakiye

Ride Bnb akaba program umuntu akoreramo amafaranga adahanayitse, atuje atibaza ngo arahomba ngo birafungwa aribyo usanga biba bihanayikisije benshi, mugukorera amafaranga Online, bibazako umutungo wabo batawigengaho kuko ufite abagenzura. Arko muri RideBNB, Umutungo abaari uwanyirawo, niwowe uwicungira ntawundi muntu ubyivanzemo, Mbese aba ari nyiri gukoresha application na system gusa


Iyi RideBNB Tukaba turimo kuyikoreramo amafaranga Twifashishije SAFEPAL, Safepal ni wallet iri mubwoko bwa nonecastodia wallet, akaba ari ubwoko bwa wallet bwemerera umuntu kwigenzurira burundu no gucuka imitungo ya cyptocurrency afite kuri iyo wallet ntawundi muntu ubijemo, Izi wallet akaba ari wallet zifungurwa n'amagambo 12 ariyo yitwa Private Keys cyangwa se Recorvery phase. Ayo akaba ari amagambo afungura wallet yawe, ni nka Password yawe niwowe uyagenzura ukayacunga.
